# cat_randomimg

버튼을 클릭할 때마다 랜덤 고양이 사진을 보여주는 간단하고 재미있는 웹 페이지입니다. HTML, CSS, JavaScript로 제작되었습니다.

## 기능

- 랜덤 고양이 사진 표시
- 반응형 디자인
- 심플하고 깔끔한 사용자 인터페이스
